# YouTube Analyzer - Frontend

This is the frontend portion of the YouTube Analyzer project. I built this using React to create a clean, interactive user interface for analyzing YouTube videos.

## Technologies Used

- **React**: The core library I used for building the UI components.
- **Axios**: Handing all the HTTP requests to communicate with our NestJS backend.
- **Lucide React**: For providing clean and consistent icons across the interface.
- **TypeScript**: To keep the code type-safe and catch errors early during development.

## Getting Started

To get the frontend up and running locally, just follow these steps:

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

The application will start up and should be available at [http://localhost:3000](http://localhost:3000). The page will automatically reload if you make any changes to the code.

## Available Scripts

In the project directory, you can run:

- `npm start`: Runs the app in development mode.
- `npm test`: Launches the test runner if you want to run or write tests.
- `npm run build`: Compiles the application for production, bundling React and optimizing the build for the best performance.
